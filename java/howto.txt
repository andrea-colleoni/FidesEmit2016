File howto.txt - tip and tricks di Java

-- Compilare file .java da linea di comando --

javac -g *.java

-- Compilare file .java da linea di comando specificando le librerie esterne .jar in uso nel programma --

javac -g -cp mysql-connector-java-5.0.8-bin.jar *.java

-- Una volta compilate, dovreste avere nella stessa cartella dei file .class --
-- Specificate la classe, senza la parte finale .class, che contiene al suo interno un metodo main, tramite il seguente comando --

java App

-- e il metodo main partirà se non incontrerà nessun errore --
-- Se avete specificato un file .jar durante la compilazione come nel secondo esempio poco sopra, dovete specificarlo anche quando eseguite la classe java col metodo main --

java -cp .:mysql-connector-java-5.0.8-bin.jar App

NOTA: È fondamentale che non dimentichiate di specificare .: dopo -cp e prima del nome del vostro file .jar, dato che non sono errori di punteggiatura.
