Introduzione

Un file HTML5 ha la seguente struttura generica

    <!DOCTYPE html>
    <html>
        <head>
            <meta charset="utf-8">
	    <title>HTML5 Tutorial (aculnaig)</title>
	</head>
	<body>
	</body>
    </html>


<!DOCTYPE html> deve essere la prima dichiarazione che fate nel vostro file HTML, ancor prima del tag <html>; NON è un tag, ma è una semplice istruzione che indica al browser web quale versione di HTML è scritto il vostro file.
